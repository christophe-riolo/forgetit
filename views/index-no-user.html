Todo.txt++

An online todo list manager based on the Todo.txt format — complete with with Dropbox integration. If you keep your todos in a text file, you've already finished the hardest part.

All code is open-source (MIT license) and available on GitHub.

Dropbox Integration

Automatically synchronize your Todo.txt file to any location in your Dropbox account. We'll pull in any update you make elsewhere in real-time.

Standards Compliant

We play by the rules. We can understand your existing Todo.txt file and whatever changes you make here will be understood by any other Todo.txt-compliant program.

Filters

Search and refine your todo list by project, context, keyword, or priority. You can keep a list of your favorite filters in the sidebar so they'll always be one click away.

Open Source

Go ahead — grab the code from GitHub. Download it, fork it, patch it, whatever. It's all yours under the MIT license. Do something interesting? Send us a pull request!

Mobile Device Support

We use a responsive layout to give you a comfortable experience whether you're on a phone, tablet, or full-size screen. Access your todos through a familiar interface wherever you are.

Raw Text Support

Sometimes you want control; we get it. Switch over to a familiar raw text view at any time to edit your todo list. We even have Todo.txt syntax highlighting.
